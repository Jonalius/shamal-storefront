## Section 1 — shamal-product-main

Layout:
- `<section>` full-width, `bg-shamal-black`, `min-h-screen`, `py-20 md:py-28`
- Inner grid: `grid-cols-1 md:grid-cols-2 gap-12 md:gap-16` inside `max-w-[1400px] mx-auto px-6 md:px-10`

Gallery (left):
- Uses local state for active image index
- `product.images.nodes` from useLoaderData (fallback to `[featuredImage]` if `images` empty)
- Hero shot: `aspect-[3/4] bg-shamal-surface`, image `object-contain`, gold radial gradient behind via absolute `bg-[radial-gradient(...)]` similar to discovery section
- Thumbnail strip below if `images.length > 1`: row of small squares, active one outlined `border-shamal-gold`

Right column:
- Gold label = `product.productType || product.tags[0] || "SHAMAL"`, uppercased
- `<h1>` Cormorant `text-4xl md:text-5xl lg:text-6xl text-shamal-white`
- Italic subtitle: editable Weaverse field; default → first non-empty line of `product.description`
- `descriptionHtml` rendered inside a dark prose wrapper — using inline classes on a wrapper div, not a `prose` plugin (the repo has no `@tailwindcss/typography` based on what we've seen; safer to define basic styles directly)
- `DiamondDivider`
- Price block: `<Money data={selectedVariant.price || priceRange.minVariantPrice}>` in `font-cormorant text-4xl text-shamal-white`
- CTA branching:
  - `isRefill` → gold button "REFILL YOUR MOMENT" + "Crafted to order" italic
    - Still uses `AddToCartButton` (it's a real purchasable product, just framed as refill)
  - `isPreOrder` (and not refill) → gold button "RESERVE YOUR VOYAGE" + ship note
    - Still uses `AddToCartButton` so checkout works
  - Else if `selectedVariant.availableForSale` → `AddToCartButton` with editable "ADD TO CART" text
  - Else → disabled "SOLD OUT" outline
- Quantity selector: integer 1–10, only rendered when neither refill nor pre-order
- Shipping note: "Free shipping within Finland & EU" — editable

Schema groups (Copy / CTA / Refill / Pre-order / Shipping). All branch copy is editable.

## Section 2 — shamal-product-details

Layout:
- Full-width, `bg-shamal-surface`, `py-20 md:py-28`
- `max-w-[860px] mx-auto px-6 md:px-10 text-center`

Content:
- Gold tracked label — editable; defaults vary by detected product variant family but the section uses one default in the schema
- Branching:
  - If voyage (matches `voyage-N` / `moment-N` tag): renders three columns TOP / HEART / BASE with editable per-column heading + body
  - Else: renders the "included items" list (Weaverse-editable, three lines by default)
- `DiamondDivider`
- "Crafted in Helsinki" small italic footer — editable

All notes are Weaverse text fields. We deliberately don't pull from metafields yet — metafield wiring is out of scope and the schema's defaults are sensible.
