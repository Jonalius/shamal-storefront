## Section 1 — shamal-collection-hero

Schema groups:
- Background: `backgroundImage` (image, optional)
- Label: `label` (text, default `"01 · THE COLLECTION"`)
- Copy: `headline` (text, empty default → fall back to `collection.title`), `subhead` (text, optional italic small), `paragraph` (textarea, empty default → fall back to `collection.description`)

Behavior:
- `useLoaderData()` typed against the collection route loader to read `collection.title` / `collection.description`
- Fallback: if a schema field is non-empty, use it; otherwise use the Shopify value
- IntersectionObserver one-shot fade-in-up, same pattern as `shamal-discovery`
- Local `DiamondDivider` matching the motif in `shamal-discovery`
- Background image + dark overlay, identical pattern to `shamal-hero` / `shamal-discovery`

## Section 2 — shamal-collection-grid

Schema groups:
- Layout: `columns` (range 1–4, default 3)
- Copy: nothing (the section is just a grid; copy lives in the hero)

Behavior:
- `useLoaderData()` reads `collection.products.nodes`
- Empty state: centered Cormorant italic "Coming later this year." + small line "Join the waitlist…"
- Otherwise: 3-col responsive grid (`grid-cols-1 md:grid-cols-3`), each card styled to match `shamal-shop-grid.tsx`
- Voyage label logic: if `product.tags` includes a tag matching `/^voyage[-\s]?(\d+)$/i` or `/^moment[-\s]?(\d+)$/i`, render `MOMENT 0X` label in gold; else omit
- Subtitle logic: first non-empty line of `product.description` truncated; else omit. (Metafield-based subtitle deferred — no metafield is currently wired up; description fallback covers the immediate need)
- CTA logic:
  - If `tags` includes `"pre-order"` → gold "RESERVE YOUR VOYAGE" + "Ships end of June" small italic line below
  - Else if `product.selectedOrFirstAvailableVariant?.availableForSale` → outlined "ORDER NOW" link to `/products/{handle}`
  - Else → disabled "SOLD OUT" treatment (no link)
- Price uses `product.priceRange.minVariantPrice` formatted with `Money` from `@shopify/hydrogen`
- Staggered reveals: header (none — no header in grid), each card delayed `index * 150ms` after a single IntersectionObserver fires
